## 1-1. 파이썬의 데이터 저장 방식
파이썬은 메모리 관리를 위해 주로 **스택(Stack)** 과 **힙(Heap)** 영역을 사용한다.
| Stack | Heap |
| ----- | ---- |
| 변수 이름(Name)| 실제 데이터(Object) |
| 객체 주소값(Reference)| |


스택 (Stack Memory)
- 용도 : 주로 함수의 호출 정보와 지역 변수(객체에 대한 참조/이름)가 저장되는 영역
- 특징 : LIFO(Last-In, First-Out)의 구조
- 변수 이름이나 식별자, 그 이름이 가리키는 객체의 메모리 주소(Reference)가 스택에 저장된다.

힙 (Heap Memory)
- 용도 : 실제 데이터(객체 자체)가 저장되는 영역, 크기가 가변적이거나 동적으로 할당되는 데이터(리스트, 딕셔너리, 큰 문자열, 사용자 정의 객체 등)이 저장된다.
- 특징 : 메모리 공간이 필요할 떄마다 동적으로 할당되며, 사용 후에는 파이썬의 가비지 컬렉터(Garbage Collector)에 의해 자동으로 해제
- 모든 데이터 값(객체)이 힙에 저장된다. 파이썬은 모든 것이 객체이므로, 정수 하나라도 실제 값은 힙에 존재한다.

## 2. 파이썬의 변수와 메모리 동작 원리
파이썬에서 변수는 **객체(데이터)를 가리키는 이름표(Lable)** 이다.

1. **객체 생성**
    - `x = 10` 정수 값 10을 가진 객체가 힙 메모리에 생성된다.
2. 이름 할당(바인딩)
    - 변수 `x`라는 이름이 스택에 생성된다.
    - 이 이름 `x`는 힙에 있는 객체 `10` 의 메모리 주소(Reference)를 저장하고, 이 객체를 가르킨다.
3. 재할당 : 
    - `x = 20` 새로운 객체 20이 힙 메모리에 생성된다.
    - 이름 `x`는 더 이상 객체 `10`을 가리키지 않고, 새로 생성된 객체 `20`을 가리키도록 참조가 변경된다.
    - 객체 `10`은 더 이상 참조하는 이름이 없으면 가비지 컬렉터에 의해 회수된다.

## 3. 불변 객체(Immutable)와 가변 객체(Mutable)란?
|특징|불변 객체(Immutable)|가변 객체(Mutable)|
|--|------------------|----------------|
|정의| 객체 생성 후 내용 변경 불가 | 객체 생성 후 내용 변경 가능|
|자료형|int, float, str, bool, tuple|list, dict, set|
|재할당 시 | 새로운 객체가 힙에 생성, 변수는 새 객체를 참조 | 객체 내부의 값만 변경,객체의 주소는 그대로 유지|
|코드 | `x = 10` `x = 20` | `y = [1, 2]` `y.append(3)`|
|메모리 변화| 10 객체는 남고, 20 객체가 새로 생성됨 | [1, 2]에 대한 주소는 유지, 내부의 값만 [1, 2, 3]으로 변경됨|

## 4. 숫자형(Numeric Type)
- `int` : 정수  
- `float` : 실수  

```Python
# 기본 연산자
a = 2
b = 4

print(a + b)  # 6 더하기 연산자
print(a - b)  # -2 빼기 연산자
print(a * b)  # 8 곱하기 연산자
print(a / b)  # 0.5 나누기 연산자 / 나누기 연산자 사용후에는 실수로 반환
print(a ** b) # 16 제곱 연산자
print(a % b)  # 2 b를 a로 나눈 후 몫(정수) 반환
print(a // b) # 0 b를 a로 나눈 후 나머지 반환


# 대입 연산자
c = 1

c += 1 # 더하기 후 대입
print(c) # 2
c -= 1 # 빼기 후 대입
print(c) # 1
c *= 1 # 곱하기 후 대입
print(c) # 1
c /= 1 # 나누기 후 대입
print(c) # 1.0


# 숫자형 데이터를 다루는 파이썬 내장 함수

abs()   # 절대값 반환
round() # 반올림, 두 번째 인자로 소수점 자리수 지정 가능
pow()   # a의 b제곱 반환
max()   # 최대값 반환
min()   # 최소값 반환
```

## 5. 문자형(String Type)
- `str` :  문자열  
- 작은 따옴표`'`, 큰 따옴표`"`로 감싸서 사용한다.
- 여러 줄에 걸쳐 사용하려면 따옴표 3개 사용

```Python
# 문자형 데이터를 다루는 파이썬 내장 함수

len()              # 문자열 길이 반환
.upper()           # 모든 문자를 소문자로 변환
.lower()           # 모든 문자를 대문자로 변환
.capitalize()      # 문장 시작을 대문자로, 나머지는 소문자로 변환
.title()           # 모든 단어의 첫 글자를 대문자로 변환
.strip()           # 문자열 양쪽 끝 공백 제거
.lstrip()          # 문자열 왼쪽 공백 제거
.rstrip()          # 문자열 오른쪽 공백 제거
.split(separator)  # 특정 구분자를 기준으로 문자열 분리 후 리스트로 반환
.replace(old, new) # 특정 문자열을 다른 문자열로 대체
.find()            # 문자열에서 찾는 내용이 없으면 -1 반환
.index()           # 문자열 찾는 내용이 없으면 오류 반환
.count()           # 문자열에서 특정 내용이 몇 번 나오는지 세어줌
.split()           # 문자열을 특정 기준으로 나눔


# 문자열에 더하기 / 곱하기 연산자 사용
string = 'Hi'

print(string + string) # HiHi
print(string * 3)      # HiHiHi
```

## 5-1. 문자열 인덱싱(Indexing)
인덱싱(Indexing)
- 문자열에서 특정 위치(인덱스)에 있는 단일 문자에 접근하는 방법
- 양수 인덱스 : 0부터 시작하여 문자열의 앞에서부터 접근한다.
- 음수 인덱스 : -1부터 시작하여 문자열의 뒤에서부터 접근한다.

```Python
# 인덱싱 예시
s = "hello world"

print(s[0])     # h
print(s[1])     # e
print(s[5])     # 
print(s[-1])    # d
print(s[-2])    # l
```
---
## 5-2. 문자열 슬라이싱(Slicing)
슬라이싱(Slicing)
- 문자열의 일부분을 잘라내어 새로운 문자열을 만드는 방법
- [시작 인덱스 : 끝 인덱스 : 간격]
- 끝 인덱스는 포함되지 않음
- 슬라이싱 결과는 항상 새로운 문자열 객체

```Python
# 인덱싱 예시
s = "hello world"

print(s[:5])    # hello
print(s[6:])    # world
print(s[0:5:2]) # hlo
print(s[::-1])  # dlrow olleh
```

참`True` 또는 거짓`False`의 논리 값을 나타내는 타입

## 6-1. Truthy와 Falsy (참/거짓 같은 값)
- 파이썬에서는 모든 객체가 boolean값으로 평가될 수 있다.
- `True` 로 간주되는 값을 Truthy 라고 한다.
- `False` 로 간주되는 값을 Falsy 라고 한다.

**Truthy**
> 0이 아닌 모든 숫자  
비어 있지 않은 모든 컬렉션 및 문자열, True

**Falsy**
> 정수 0, 실수 0.0  
비어 있는 모든 컬렉션 과 빈 문자열 [], {}, (), ""  
None, False
