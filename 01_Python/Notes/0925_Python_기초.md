## 1. 프로그래밍(Programming)이란?
> 컴퓨터에게 우리가 원하는 작업을 수행하도록 지시를 내리는 행위  
프로그래밍은 컴퓨터가 이해할 수 있는 언어(프로그래밍 언어)를 사용하여 특정 순서와 논리에 따라 명령을 작성하는 과정
---
## 1-1. 컴파일러(Compiler) vs 인터프리터(Interpreter)
컴파일러와 인터프리터는 모두 고급언어로 작성된 소스 코드를 컴퓨터가 이해할 수 있는 **기계어로 번역하는 역할**을 수행한다.
| 특징    | 컴파일러 (Compiler) | 인터프리터 (Interpreter) |
| -------| ----------------- | --------------------- |
| 번역 시점 | 프로그램을 전체 실행하기 전에 한 번에 번역| 프로그램을 실행하는 동안 한 번에 한 줄씩 번역하고 실행 |
| 출력 파일 | 번역 후 독립적인 실행 파일 생성| 별도의 실행 파일을 생성하지 않고 즉시 실행|
| 소스 코드 |실행 시 원본 소스코드가 필요하지 않음 |실행시 원본 소스코드가 매번 필요 |
| 실행 속도 |번역 과정이 길지만, 한 번 컴파일된 후에는 실행속도가 빠름 | 번역 과정은 짧지만, 실행할 때마다 번역하므로 실행속도가 느림|
| 오류 발견 |컴파일 단계에서 전체 오류를 한 번에 보고 |실행 중 해당 줄에 도달했을 때 줄별 오류 보고 |
| 대표 언어 | C, C++|Python, JavaScript |

## 2. 파이썬(Python)이란?
1991년 귀도 반 로섬에 의해 개발된 고급(High-level) 프로그래밍 언어
- **높은 가독성 및 간결성** : 문법이 직관적이고 간결하여 가독성이 높음, 들여쓰기로 코드블록을 구분함
- **인터프리터 언어** : 컴파일 과정이 없어 코드를 수정하고 결과를 바로 확인하기 쉬움
- **동적 타이핑** : 변수 선언 시 자료형을 미리 지정할 필요 없음
- **풍부한 라이브러리 생태계** : AI, 데이터, 웹, 자동화 라이브러리 생태계가 풍부하다

## 3. 아나콘다(Anaconda)란?
아나콘다는 파이썬과 R 언어를 기반으로 하는 데이터 과학 및 머신러닝을 위한 통합 배포판이자 패키지 관리 시스템
- **통합환경** : 파이썬 자체뿐만 아니라 데이터 분석에 필요한 NumPy, Pandas, Scikit-learn, Jupyter Notebook 등의 핵심 라이브러리 및 도구를 미리 설치된 상태로 제공한다.
- **패키지 관리** : `conda` 패키지 관리자를 통해 쉽게 라이브러리 설치, 업데이트, 제거가 가능하다.
- **가상 환경 관리** : 가상 환경을 쉽고 안정적으로 생성 및 관리할 수 있다.

## 4. Python 입력과 출력
입력(Input)
- `input()` 함수 사용
- 키보드로 입력을 받아 **문자열**로 저장되기 떄문에 숫자가 필요한 경우 형 변환 필요 (`int()`, `float()`)
```Python
name = input('이름을 입력하세요: ')
age = int(input('나이를 입력하세요: '))
print('안녕하세요,', name, '입니다.')
print('내년에는', age+1, '살이 됩니다!')

# 이름을 입력하세요: 철수
# 나이를 입력하세요: 20
# 안녕하세요, 철수 입니다.
# 내년에는 21 살이 됩니다!
```

---
출력(Output)
- `print()` 함수 사용
- 콤마`,`로 여러개 나열 가능 (자동으로 띄어쓰기 한 칸 삽입)

```Python
print('Hello World!') # Hello World!
print('이름:', '철수') # 이름: 철수
```

출력 옵션
- `sep` : 구분자 설정
- `end` : 줄 끝 문자 설정
```Python
print('A', 'B', 'C', sep='-')
print('Hello', end=' ')
print('World')
# A-B-C
# Hello World
```

출력 서식 (f-string)
```Python
name = '철수'
age = 20
print(f'안녕하세요, {name}입니다. 나이는 {age}살 입니다.')
# 안녕하세요, 철수입니다. 나이는 20살 입니다.
```

출력 서식(format)
```Python
name = '철수'
age = 20
print(f'안녕하세요, {}입니다. 나이는 {}살 입니다.'.format('영희', 20))
```
