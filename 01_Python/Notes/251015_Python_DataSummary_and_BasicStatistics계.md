## 1. 평균(Mean)
- 데이터의 모든 값을 더한 후, 데이터의 개수로 나눈 값
- 데이터의 '무게 중심' 역할을 한다.
- 모든 데이터를 반영하지만, 너무 크거나 작은 값(이상치, Outlier)에 영향을 많이 받는다.
```Python
import numpy as np

scores = np.array([80, 90, 100, 70, 60])
print(f'평균 점수 : {np.mean(scores)}') # 평균 점수 : 80.0
```

## 2. 중앙값(Median)
- 데이터를 크기 순서대로 나열했을 때, 가장 한가운데 있는 값
- 데이터의 '허리' 역할을 한다.
- 극단적으로 크거나 작은 값이 있어도 값이 크게 변하지 않는다.
- 소득 수준(연봉 등)처럼 격차가 큰 데이터를 분석할 때 평균보다 유용하다.

```Python
import numpy as np

salaries = np.array([3000, 3200, 3100, 8000])
print(f'평균 연봉 : {np.mean(salaries)}')    # 평균 연봉 : 4325.0
print(f'중앙값 연봉 : {np.median(salaries)}') # 중앙값 연봉 : 3150.0
```

## 3. 표준편차(Standard Deviation)
- 데이터가 평균으로부터 얼마나 멀리 떨어져 있는지(퍼져 있는지)를 나타내는 수치
- 데이터의 '불확실성' 또는 '변동성'을 보여준다.
- **값이 작으면**: 데이터들이 평균 주변에 모여 있음 (안정적)
- **값이 크면**: 데이터들이 넓게 퍼져 있음 (변화가 심함)

> 예시)  
양궁 선수 1 (표준편차 작음) : 쏠 때마다 9점, 10점 근처에 화살이 모인다. 안정적인 선수!  
양궁 선수 2 (표준편차 큼) : 10점, 5점, 3점 화살이 과녁 전체에 흩어져 있다. 기복이 심한 선수!

```Python
import numpy as np

player1_score = np.array([9, 10, 9, 10])
player2_score = np.array([10, 5, 3, 6])

print(f'선수1 표준편차 : {np.std(player1_score):.2f}') # 선수1 표준편차 : 0.50
print(f'선수2 표준편차 : {np.std(player2_score):.2f}') # 선수2 표준편차 : 2.55
```

## 4. 히스토그램(Histogram)
- 연속적인 데이터를 일정 구간으로 나누어, 각 구간에 속한 데이터의 개수(빈도)를 막대그래프로 나타낸 것
- 데이터가 어디에 집중되어 있고, 전체적인 분포가 어떤 모양인지 한눈에 확인하기 위해 사용한다.
- `plt.hist(data)` 또는 `sns.histplot(data)` 을 사용해 시각화 할 수 있다.

## 4-1. 히스토그램의 모양과 데이터 성격
예쁜 종 모양 (정규분포, 평균 = 중앙값)  
- 모양: 좌우가 대칭이며 가운데가 볼록한 형태
- 해석: 평균, 중앙값, 최빈값이 거의 일치하며 분석하기에 가장 이상적인 형태입니다.

뾰족한 모양 (표준편차 작음)  
- 모양: 가운데가 아주 높게 솟아 있고 옆으로 퍼진 부분이 적은 형태  
- 해석: 데이터가 평균 근처에 아주 밀집되어 있다.

완만한 모양 (표준편차 큼)
- 모양: 옆으로 펑퍼짐하게 퍼져 있는 형태
- 해석: 데이터가 넓은 범위에 골고루 퍼져 있다.

오른쪽 꼬리가 긴 모양 (평균 > 중앙값)
- 모양: 왼쪽(작은 값)에 데이터가 몰려 있고, 오른쪽으로 꼬리가 긴 형태
- 해석: 대다수는 작은 값을 가지지만, 극소수의 아주 큰 값이 존재하는 경우이다.
