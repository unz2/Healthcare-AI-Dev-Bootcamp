## Projects

### 1. Healthcare Data Prediction Model

**헬스케어 데이터를 활용한 예측 모델 개발 프로젝트**

- **목표**
  - 실제 헬스케어 데이터를 분석하고 예측 문제를 머신러닝 모델로 해결

- **주요 작업**
  - 데이터 전처리 및 탐색적 데이터 분석(EDA)
  - 머신러닝 모델 학습 및 성능 비교
  - 결과 분석을 통한 모델 개선 시도

- **사용 기술**
  - Python, Scikit-learn

- **핵심 성과**
  - 헬스케어 데이터 기반 예측 파이프라인 구축 경험
  - 데이터 특성에 따른 모델 선택 및 평가 역량 강화

### 2. Smoking Status Prediction AI Hackathon

**흡연 여부 예측 AI 해커톤 (팀 프로젝트)**

- **목표**
  - 개인 건강 데이터를 기반으로 흡연 여부를 예측하는 AI 모델 개발

- **환경**
  - 제한된 시간 내 문제 해결이 요구되는 해커톤 형태

- **주요 작업**
  - 데이터 분석 및 주요 Feature 선정
  - 머신러닝 / 딥러닝 모델 설계
  - 성능 개선을 위한 실험 및 튜닝

- **사용 기술**
  - Python
  - Machine Learning / AutoGluon

- **의의**
  - 실전 문제 해결 중심의 팀 협업 경험
  - 시간 제약 환경에서의 모델 성능 최적화 경험
